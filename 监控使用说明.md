## 🚀 使用方法

### 方式一: 浏览器动态监控（推荐）

**已自动打开的监控页面: http://localhost:8000/**

**功能特性:**
- 🔄 **立即刷新**: 手动触发数据抓取
- ⏰ **自动刷新**: 可设置30秒/1分钟/5分钟/10分钟/30分钟自动更新
- 📊 **实时统计**: 显示匹配关键词的热搜数量
- 📈 **历史记录**: 自动保存匹配到的历史数据
- 🎨 **高亮显示**: 自动高亮关键词

**操作步骤:**
1. 点击"立即刷新"按钮 - 抓取最新抖音热搜
2. 点击"开启自动刷新" - 启动定时监控
3. 在下拉菜单选择刷新间隔

### 方式二: 手动运行

在项目目录下运行:
```bash
cd TrendRadar
python main.py
```

生成的数据保存在: `output/2025年11月13日/`

### 方式三: 查看静态报告

打开文件:
```
TrendRadar/output/2025年11月13日/html/当日汇总.html
```

## 🛠️ 故障排查

### 问题1: 服务器无法启动
**解决方案:**
```bash
cd TrendRadar
python monitor_server.py
```

### 问题2: 端口被占用
**解决方案:**
修改 `monitor_server.py` 最后一行:
```python
run_server(port=8001)  # 改为其他端口
```

### 问题3: 数据不更新
**检查步骤:**
1. 确认服务器正在运行
2. 检查网络连接
3. 手动运行 `python main.py` 测试
